Getting Started with Gallery Pro

Gallery Pro is a flexible, beautiful showcase-style WordPress theme with a very flexible layout, the ability to change thumbnail sizes, post videos, and more!

Skip to Installation Instructions

Features

Gallery Pro offers some robust but easy-to-use features:

  • Two Beautiful Color Schemes

    In addition to the original Gallery dark theme, there is now a light theme as well, which can be enabled on the Gallery Pro Options page.

  • Alternate Blog Post Template

    By popular demand, there is now an alternate blog post template that can be enabled on a per-post basis. Simply edit the post, scroll to the bottom of the edit page, and quickly and easily switch the page template.

  • Simple Video Embedding

    Included with the original Gallery theme, the ability to share videos using the Simple Video Embedder plugin from Press75.com is still here and the videos have been resized to better fit the theme. Requires installation of Simple Video Embedder plugin, download separately.

  • iPhone Template

    Located in the content area of the index, or blog page. Empty of widgets by default, they will not apear in the theme markup until you add a widget. The location of the Index Insert, which appears between posts, is controlled from the theme options page. The default position sets it to appear after the second post.

  • Resizable Thumbnails

    No longer fixed at 125x125, thumbnail sizes are completely under your control! Just set your desired thumbnail size in the Gallery Pro Options page.

  • Automatic Image Resizing

    Gallery Pro now uses TimThumb by Darren Hoyt to resize thumbnails and images within posts. That means all you need to do is upload at least a large image for each post and you're ready to go. No more manual image resizing/multiple uploads.

  • Image Gallery with Lightbox

    Now a built-in feature of Gallery Pro, simply upload full-size images with the default WordPress image uploader and Gallery Pro will automagically add a gallery to your post with a lightbox for large images!

  • WP-PageNavi Built-in

    Another new feature, WP-PageNavi is now built-in to Gallery Pro, offering easy page navigation for all!

  • Ad Zones Built-in

    There are now two ad zones that accept plain HTML and Javascript below the header and above the footer.

  • New! Add a sidebar to your Homepage and Category Pages

    There is a new option to show your sidebar on the homepage and category pages along with the ability to control the width of the sidebar as well, giving you extra control over the layout of Gallery Pro.

Installing Gallery Pro

  1. Unzip the Gallery Pro ZIP file to your My Documents folder.
  2. Upload /gallerypro/ folder all files within to your /wp-content/themes/ directory using an FTP program.
  3. Using DreamWeaver or FileZilla (or a similar program that allows you to change remote folder permissions) CHMOD the following folders to 777:
    • /gallerypro/uploads/
    • /gallerypro/timthumb/cache/
  4. In your WordPress admin, visit the UpThemes Framework theme options page and set up the various options you require.
  5. After that's done, go to Settings > Reading and set "Blog pages to show at most" to a number that makes your images end at the end of the last row of thumbails. For three across, I recommend 12 or 15, for 4 across, 12 or 16.
  6. And you're off! Start building your gallery!

Additional Instructions

Adding Gallery Items

To add gallery items, all you need to do is create a new post, upload an image greater than 500x375 pixels, select a category, and publish the post. DO NOT insert the image into the body of your post. This will result in an image on the right side of the page. If this is what you want, then ignore this warning. To create a lightbox image slider for the post, simply add multiple images to the post. Again, you do not need to post the image in the body of your post for Gallery Pro to do its thing.

Disabling Timthumb and using the WordPress Media Library

Note: When changing media sizes in your settings, WordPress will NOT resize images that have already been added. You will want to install the Regenerate Thumbnails plugin, which resizes all your images to the correct WordPress media sizes if you've recently changed them.

If you want to avoid using Timthumb and just use the media library settings provided by WordPress, which is completely acceptable (and recommended), you need to follow the following instructions to set it up.

Please note that if you are upgrading from an older version of Gallery or Gallery Pro and turn off timthumb, your images will most likely be the wrong sizes. This option is recommended mainly for new sites using the Gallery Pro theme, as they will start out with no images.

Once you've decided what size thumbnails to use, you should go into Settings/Media and set your thumbnail size to the size you've defined in your Gallery Pro Options page then set your medium size to 500x375, and large size to 1000x1000 or something in that range.

Create a User Submission Page

Note: I cannot provide an import file for this step because every install is different and I cannot create an import file that everyone can use.

  1. Install the TDO Mini-Forms plugin. Activate it.
  2. Go to the Form Creator from the TDO Mini-Forms menu. Add the widgets to and from the form that you require users to fill out when submitting. My form contains the following:
    • Upload Files 1 (adds a file upload field)
    • Custom Fields 1 (on options page for this item - Title: Designer Name - Custom Field Key: designed-by)
    • Custom Fields 2 (on options page for this item - Title: Website URL - Custom Field Key: web-url)
    • Content
    • Categories 1
    • Tags
    • Notify Me
  3. Add the shortcode for your form to one of your WordPress pages or use TDO Mini-Forms admin to assign a page for this.
  4. Styling the page is not easy. TDO Mini-Forms has its own CSS file for the file uploader. Try editing the file tdomf-style-form.css in that plugins folder to achieve the right CSS for that piece.
  5. If you need any support for this form, please visit the TDO Mini-Forms support site.

Add Star Ratings

  1. Install the WP-Post Ratings plugin from WordPress.org. Activate both the plugin and the widgets.
  2. To switch the type of images to transparent .PNGs, open wp-content/plugins/wp-postratings/wp-postratings.php in a text editor and change the following lines (should be near the top of the document):

    ### Define Image Extension
    define('RATINGS_IMG_EXT', 'gif');
    //define('RATINGS_IMG_EXT', 'png');

    to match this:

    ### Define Image Extension
    //define('RATINGS_IMG_EXT', 'gif');
    define('RATINGS_IMG_EXT', 'png');

  3. Then upload that file back into the plugins folder.
  4. In your WordPress admin panel, click the Ratings > Options link on the left sidebar.
  5. The transparent .PNG stars should be selected by default.
  6. Under 'Ratings AJAX Style' change 'Show Loading Image With Text' to 'No' and 'Show Fading In And Fading Out Of Ratings' to 'No.'
  7. Click 'Save Changes.'
  8. Now click the 'Ratings Templates' on the left sidebar.
  9. Replace the text in the 'Ratings Vote Text' box with the following code:

    %RATINGS_IMAGES_VOTE%

  10. Replace the text in the 'Ratings Voted Text' with the following code:

    %RATINGS_IMAGES%

  11. Click the 'Save' button.
  12. You should now see your star ratings show on the gallery single pages next to the title of the gallery item.

Using the Custom iPhone Post Template

Note: Requires Custom Post Template plugin.

  1. Create a new post or Edit an old one.
  2. Upload an image to that specific post, making sure the image is sized at 223x334.
  3. Copy the location of that image.
  4. Paste the image path into the "full-size image" field in the Custom Post Settings.
  5. In the Post Template widget, select the iPhone template.
  6. Publish the post.

Using the Custom Blog Post Template

Note: Requires Custom Post Template plugin.

Gallery Pro comes with a blog template that can be used for normal blog entries.

  1. Create a new blog post or edit an existing one.
  2. In the Post Template widget, select the Blogpost template.
  3. Publish the post.

Adding Advertisements to your Site

Using the Built-In Header and Footer ad zones (recommended)

  1. Go to Gallery Pro Options.
  2. Enter the HTML or Javascript from your ad company into the Ads Below Header or Ads Below Footer area.
  3. Click Save changes.

Using Widgets

  1. Go to Widgets.
  2. On the right side, there will be a set of widget zones for Index, Single, and Page. This allows you to add widgets to different sections of your site
  3. From the list of widgets, select a Text widget and drag it over to the zone you want the ads to be shown in.
  4. Enter the ad code into the textbox and save it.

Adding Widgets to the Footer

  1. Make sure you activated the WP-Post Ratings widgets when you activated the plugin.
  2. In your WordPress Admin, click Widgets on the sidebar.
  3. From the drop-down box under 'Current Widgets' select '1st Subsidiary Aside' and click 'Show.'
  4. Drag the widgets you'd like to see shown in this area from left to right (Highest Rated, Most Rated, Recent Comments, etc.)
  5. Click 'Save.'
  6. You should now see your widgets in the footer.

Using Videos in Posts

Gallery Pro works with the Simple Video Embedder plugin from Press75.com.

  1. Install and activate the Simple Video Embedder plugin.
  2. Go to the Video Options (on the left under Settings) page and set them to a default width of 500 and height of 306.
  3. On the post page, simply enter the URL of the video into the Video URL box (in the Post Video Options widget).
  4. Click Update Post.

Troubleshooting

Trouble with Timthumb/Images Not Displaying Properly

Images Must Reside on your Server

You cannot use images hosted on Amazon S3 or on another server, they must be local, otherwise the script will not work.

Cache Folder Must Have Permissions set to 755 or 777 (if allowed by host)

Some webhosts do not allow 777 but you do need to have the /gallerypro/timthumb/cache folder chmodded to 755 at the very least.

Timthumb Does Not Work Locally

If you plan to test locally, you will not be able to view images using timthumb.

If It Still Doesn't Work Right

If you still have problems with images loading after ensuring all these steps have been taken, there is a setting on the Gallery Pro Theme Options page that allows you to turn Timthumb off completely. If you use this option, you MUST set up your thumbnails and medium-size images to the correct sizes for them to work properly. These can be set in Settings > Media. You need to set your thumbnail size to the exact size of the thumbnail size you set in the Gallery Pro Options page and set the medium size to 500x375.
